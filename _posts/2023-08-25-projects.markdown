[This](https://star-study.web.app/) is a severely broken studying application focused on a unique method of studying. It uses Firebase for auth and the database.

Currently i'm building a simple in progress 3d renderer built using SFML and c++ because I wanted to challenge myself.

### Prompty

[This](https://prompty-bff99300b53b.herokuapp.com/) project was built in three days for [Outernet](https://outernet.hackclub.com/) and is meant to help people excersise prompt engineering. You can play it if the application is up, but it probably won't be.


### C++ Raycasing Engine
[This](https://github.com/Captainexpo-1/cpp-raycasting-engine) is the second raycasting engine that I've wrote, it's in c++ and uses SFML as the graphics library. Currently it has a half-functioning texture system and a broken sprite/entity system, but it helped me learn the basics of the c++ language.

### Sinerider Contributions

I contributed to the making of [this game](https://sinerider.com) near the end of development, but it was the first time that I worked in and with "real" team. I was mainly doing UI work and fixing bugs on occasion, but it really helped me learn how to communicate and work with someone other than myself.
